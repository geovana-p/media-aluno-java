# calculador simples da media de um aluno em java

###  Funcionalidades
- calcular a media do aluno a partir de tres notas
- se a nota não estiver entre 0 e 10 ele ira pedir novamente
- informara se o aluno foi aprovado, se esta de recuperação ou reprovado
- repete o processo ate você dijitar 'sair'

### conceitos aplicados
- variaveis e tipos de dados
- entrada de dados com a classe 'scanner'
- estrutura condicionais: if-else
- estruturas de repetição: for e do-wilhe
- operadores relacionais e logicos
